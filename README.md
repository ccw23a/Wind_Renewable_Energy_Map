# Wind_Renewable_Energy_Map
This Project Repository is for assessing basic climate and current reneable energy generation in Republic of Korea using Python.

Py_Renewable, 인하대학교 에너지자원환경연구실, Choi Chan Woo

Dataset used:

https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=overview : ERA5 hourly data on single levels from 1940 to present
